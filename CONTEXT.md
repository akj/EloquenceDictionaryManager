# Eloquence Dictionaries

This context describes the dictionary content shared by the Eloquence Dictionaries add-on and supporting synthesizer add-ons.

## Language

**Managed Dictionary Set**:
A named collection of community-maintained ECI pronunciation entries made available for a synthesizer to select.
_Avoid_: Downloaded dictionaries, dictionary source

**Personal Dictionary Overlay**:
The user-owned ECI pronunciation entries that supplement or override a selected Managed Dictionary Set.
_Avoid_: User dictionaries, custom dictionaries
